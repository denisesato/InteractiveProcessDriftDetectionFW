# To install pygraphviz in a windows 64 bits machine
# It is not necessary if you are using the docker

# You can download the windows binaries for different versions of python and pygraphviz at:
https://www.lfd.uci.edu/~gohlke/pythonlibs/#pygraphviz

# In the project's folders there are the binaries for pygraphviz 1.7
# and for python versions 3.7, 3.8, and 3.9

Step 1: Download and install Graphviz

https://graphviz.gitlab.io/_pages/Download/Download_windows.html

Step 2: Check if the Graphviz installation path (bin folder) is correctly set into your PATH environment variable

Step 3: Re-open command line and activate venv in your project, example:

venv\Scripts\activate

Step 4. Install whl into your virtual environment

For example:

In case of python 3.7

pip install pygraphviz-1.7-cp37-cp37m-win_amd64.whl

In case of python 3.8

pip install pygraphviz-1.7-cp38-cp38-win_amd64.whl

In case of python 3.9

pip install pygraphviz-1.7-cp39-cp39-win_amd64.whl
